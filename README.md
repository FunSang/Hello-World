### 分支操作

`git branch` —— 显示分支一览表

```shell
git checkout -b feature-A # 创建名为 feature-A 的分支  等价于下方两条命令
## 创建 feature-A 分支，并将当前分支切换为 feature-A 分支
git branch feature-A
git checkout feature-A
```
